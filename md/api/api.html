    # 获取多本书籍信息

    • 接口地址: /book/books
    • 请求方式: POST
    • 接口描述: 获取多本书籍信息
    请求参数 类型 是否必须 描述
    subbook64s list[string] 是 数组中含有多个书籍subbook64
    language string 是 语言标识
    点击查看 POSTMAN 返回

    # 根据分类获取相关书籍

    • 接口地址: /book/category/
    • 请求方式: POST
    • 接口描述: 根据分类获得书籍
    请求参数 类型 是否必须 描述
    sub_category_int int 是 子分类id
    page int 是 第几页，page 从 1 开始
    limit int 是 一页有几个书籍
    language string 是 语言标识
    点击查看 POSTMAN 返回

    # 获取一本书籍的详细信息

    • 接口地址: /book/detail/
    • 请求方式: POST
    • 接口描述: 获取一本书籍的详细信息
    请求参数 类型 是否必须 描述
    subbook64 string 是 书籍subbook64
    language string 是 语言标识

    获取书籍的详细信息，包括章节标题，标题章节放在 chapters 字段中

    点击查看 POSTMAN 返回

    # 获取某一本书的章节信息

    • 接口地址: /book/chapters/
    • 请求方式: POST
    • 接口描述: 获取某一本书的章节信息
    请求参数 类型 是否必须 描述
    subbook64 string 是 书籍subbook64
    sort_number int 是 第几章，是章节的 sort_number 的值
    limit int 是 从 sort_number 开始，一共获得多少章
    language string 是 语言标识
    点击查看 POSTMAN 返回

    # 根据title查询书籍

    • 接口地址: /search/books
    • 请求方式: POST
    • 接口描述: 根据title查询书籍
    请求参数 类型 是否必须 描述
    title string 是 要查询的字段
    page int 是 书页，从 1 开始
    limit int 是 每页有多少本书
    language string 是 语言标识
    点击查看返回

    返回的内容参考接口 /book/books
